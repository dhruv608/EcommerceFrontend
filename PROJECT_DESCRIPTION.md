# 🛍️ Light Store - Premium E-Commerce Platform

**A modern, full-featured e-commerce platform built with cutting-edge technologies and exceptional user experience.**

## 🌟 Overview

Light Store is a sophisticated e-commerce solution that combines beautiful design with powerful functionality. Built with a microservices architecture, it offers seamless shopping experiences for customers and comprehensive management tools for administrators.

## 🎯 Key Features

### 🛒 **Customer Experience**
- **Smart Shopping Cart** - Real-time cart management with guest support and persistent sessions
- **Advanced Product Discovery** - Intelligent search, filtering, and categorization
- **Secure Checkout Process** - Multi-step checkout with multiple payment integrations
- **User Account Management** - Profile management, order history, and wishlist functionality
- **Responsive Design** - Mobile-first approach with flawless cross-device compatibility

### 🏢 **Admin Dashboard**
- **Comprehensive Analytics** - Sales insights, customer metrics, and performance tracking
- **Product Management** - Full CRUD operations with image uploads and inventory tracking
- **Order Processing** - Complete order lifecycle management from placement to fulfillment
- **Customer Management** - User administration with role-based access control
- **Category Organization** - Hierarchical product categorization and management

### 🔧 **Technical Excellence**
- **Microservices Architecture** - Scalable backend with Spring Boot
- **Modern Frontend** - Next.js 14 with App Router and server-side rendering
- **Real-time Features** - Live cart updates and order status tracking
- **Security First** - JWT-based authentication with role-based authorization
- **Performance Optimized** - Lazy loading, caching, and optimized database queries

## 🛠️ Technology Stack

### 🎨 **Frontend**
- **Next.js 14** - React framework with App Router and server components
- **TypeScript 5** - Type-safe development with enhanced IntelliSense
- **TailwindCSS 3** - Utility-first CSS framework for rapid UI development
- **React Hook Form** - Advanced form validation and state management
- **Zustand 4** - Lightweight state management solution
- **Framer Motion** - Smooth animations and micro-interactions

### ⚙️ **Backend**
- **Spring Boot 3** - Enterprise-grade Java framework
- **Spring Security 6** - Comprehensive authentication and authorization
- **Spring Data JPA** - Efficient database access with ORM
- **MySQL 8** - Reliable relational database with optimized queries
- **Redis 7** - High-performance caching and session storage
- **JWT Authentication** - Secure token-based authentication system

## 🎨 Design Philosophy

Light Store follows a **design-first approach** with:

- **Consistent Theme** - Olive gold (#ACAC49) brand color throughout
- **Accessibility Focus** - WCAG 2.1 compliant with semantic HTML
- **Performance Priority** - Optimized images, lazy loading, and minimal bundle sizes
- **Mobile-First Design** - Responsive layouts that work on all devices
- **Component-Based Architecture** - Reusable UI components for maintainability
